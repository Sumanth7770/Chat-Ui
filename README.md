# Chat UI

A modern, glassmorphic-style Chat UI built with HTML, CSS, and JavaScript (no frameworks).

## 🌟 Features

- 🎨 Theme Toggle (Cool ↔ Warm)
- 💬 Typing Indicator
- 🔍 Live Chat Search
- 📋 Context Menu (Copy, Delete, Forward)
